# Démarrage du projet Smart Grid

## Prérequis
- Docker installé
- Gradle disponible (ou utiliser le wrapper fourni)

## Commandes à exécuter

### 1. Se placer dans le dossier du projet
```bash
cd smart_grid
```
### 2. Démarrer les containers Docker
```bash
docker compose up -d
```
### 3. Lancer le backend avec Gradle
```bash
./gradlew.bat run
```
